CROPTAILS

A small project that I made is a cozy 2D farming RPG where you manage a small plot of land and explore the surroundings.

Key Features
- Core Mechanics: Planting, watering, and harvesting crops
- Systems: Day/Night cycle and Inventory management
- Art Style: Custom pixel art assets

Built With
- Engine: Godot 4.x
- Language: GDScript

How to Run
1. Download this repository as a ZIP or clone it.
2. Open **Godot Engine** (Version 4.x recommended).
3. Click **Import** and select the file.
4. Press `F5` to play!
